# BMR-Calculator-Java
A sleek and efficient desktop application designed to help users determine their Basal Metabolic Rate (BMR) and daily calorie requirements.By integrating physical parameters like weight, height, age, and activity levels, this tool provides accurate health insights to assist in fitness planning and weight management.
